개인정보처리방침

최종 업데이트: 2024년

1. 수집하는 정보

이 확장 프로그램은 사용자의 개인정보를 수집하지 않습니다.

이 확장 프로그램은 다음과 같은 정보를 수집하지 않습니다:

  • 개인 식별 정보
  • 브라우징 기록
  • 위치 정보
  • 기타 개인정보

2. 정보 사용 및 저장

이 확장 프로그램은 모든 처리를 사용자의 로컬 브라우저에서만 수행합니다. 어떠한 정보도 외부 서버로 전송되지 않으며, 서버에 저장되지 않습니다.

3. 데이터 공유

이 확장 프로그램은 사용자 데이터를 수집하지 않으므로, 제3자와 공유할 데이터가 없습니다.

4. 기능 설명

이 확장 프로그램은 용인시 공공체육시설 예약 페이지에서만 동작하며, 다음 기능을 제공합니다:

  • 동의 체크박스 자동 선택
  • 이용목적 자동 입력
  • 보안문자 자동 입력

모든 기능은 사용자의 브라우저에서만 실행되며, 어떠한 정보도 외부로 전송되지 않습니다.

5. 쿠키 및 추적 기술

이 확장 프로그램은 쿠키를 사용하지 않으며, 사용자를 추적하는 기술을 사용하지 않습니다.

6. 보안

이 확장 프로그램은 사용자 데이터를 수집하지 않으므로, 저장된 데이터에 대한 보안 문제가 없습니다. 모든 처리는 사용자의 로컬 브라우저에서만 이루어집니다.

7. 연락처

개인정보처리방침에 대한 문의사항이 있으시면, 크롬 웹 스토어의 확장 프로그램 페이지를 통해 문의해 주세요.

8. 개인정보처리방침 변경

이 개인정보처리방침은 변경될 수 있으며, 변경사항은 이 페이지에 게시됩니다. 중요한 변경사항이 있는 경우 사용자에게 알려드리겠습니다.
